
## Criar arquivo de teste Markdown

### O que será feito

Criar um arquivo `teste.md` na raiz do projeto com o conteúdo:

```
ola teste
```

### Localização

O arquivo será criado em:

```
/teste.md
```

Seguindo a estrutura existente de outros arquivos `.md` na raiz do projeto, como `README.md` e os arquivos dentro de `docs/`.
